üìú 2. MongoDB Configuration Deep Dive

The mongod.cfg file controls MongoDB's behavior.

  • storage.dbPath ‚Üí Database file location
  • systemLog.path ‚Üí Log file location
  • net.port ‚Üí Listening port (default: 27017)
  • net.bindIp ‚Üí Allowed network interfaces (use 0.0.0.0 for all)
  • security.authorization ‚Üí Enable authentication (enabled)
Example with authentication: 
security:
  authorization: enabled

üõ† 3. Backup & Restore

mongodump:

mongodump --db=mydb --out=D:\backup

mongorestore:

mongorestore --db=mydb --drop D:\backup\mydb

‚ö° 4. mongoimport & mongoexport

Import JSON:

mongoimport --db=mydb --collection=users --file=users.json --jsonArray

Import CSV:

mongoimport --db=mydb --collection=users --type=csv --headerline --file=users.csv

Export JSON:

mongoexport --db=mydb --collection=users --out=users.json --jsonArray

üîç 5. Indexing & Performance

Indexes speed up queries.

Create Index:

db.users.createIndex({email: 1})

Compound Index:

db.orders.createIndex({customerId: 1, date: -1})

Check Query Plan:

db.users.find({email: "test@example.com"}).explain("executionStats")

üóÑ 6. Data Modeling Best Practices

  • Embed documents when data is accessed together frequently.
  • Reference documents when data is large or reused often.
  • Keep document size < 16MB.
Example: 
// Embedding
{
  _id: 1,
  title: "Post",
  comments: [{text: "Nice!"}, {text: "Great!"}]
}

// Referencing
{
  _id: 1,
  title: "Post"
}
{
  postId: 1,
  text: "Nice!"
}

üöÄ 8. Connecting MongoDB to Applications

Node.js Example (Mongoose):

const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost:27017/mydb')
  .then(() => console.log("Connected!"))
  .catch(err => console.error(err));

Python Example (PyMongo):

from pymongo import MongoClient
client = MongoClient("mongodb://localhost:27017/")
db = client.mydb
print(db.list_collection_names())
